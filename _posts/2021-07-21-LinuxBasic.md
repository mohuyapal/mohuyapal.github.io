Top 10 basic handy commands:

1.whoami

It displays the username.

Syntax:
```
Mahua@HP~: whoami
```
Output:
```
Mahua
```

2.who

Syntax:
```
Mahua@HP~: who
```
Output:
```
root     pts/0        2020-11-17 20:10 (10.10.0.2)
Mahua    pts/1        2020-11-17 20:11 (10.10.0.8)
```
